# code-with-errors
Python debugging exercise containing one syntax error and one logic error for code analysis practice.
This repository contains a Python file designed for practicing reverse engineering, debugging, and code analysis.
The file includes two intentional errors:

One syntax error

One logic error

The purpose of this exercise is to encourage students to carefully read and analyze the code, understand the role of each line, and correct the existing issues.
This file is suitable for developing foundational Python skills and strengthening early-stage debugging abilities.

این مخزن شامل یک فایل پایتون است که برای تمرین مفاهیم «مهندسی معکوس»، «اشکال‌یابی»، و «تحلیل کد» طراحی شده است.
در این فایل دو خطای عمدی قرار داده شده است:

یک خطای نحوی (Syntax Error)

یک خطای منطقی (Logic Error)

هدف از این تمرین، تشویق هنرجویان به بررسی دقیق کد، تحلیل عملکرد هر خط و اصلاح اشکالات موجود است.
این فایل مناسب برای یادگیری مفاهیم پایه برنامه‌نویسی پایتون و تقویت مهارت خطایابی در سطح مقدماتی است.
